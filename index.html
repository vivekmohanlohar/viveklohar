04. Projects

  • MATLAB/Simulink

    Improvements in Transient Stability Using FACTS Devices

    Simulated the IEEE 9-Bus System in MATLAB Simulink to study transient stability and see how FACTS devices like SVC and STATCOM respond during disturbances. Compared voltage stability and power flow with and without compensation to understand how these controllers actually help the grid recover faster.

    • MATLAB Simulink
    • IEEE 9-Bus System
    • FACTS Controllers (SVC, STATCOM)
    • Power System Stability Analysis

    Built a small Arduino-based setup to detect meter bypassing and unauthorized power tapping. The Arduino reads current values off the meter line and flags irregular consumption patterns that point to theft — a real problem in distribution networks back home.

    • Arduino
    • Arduino IDE
    • Energy Distribution System
    • C++
