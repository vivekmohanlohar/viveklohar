  • MATLAB/Simulink | Currently Working

    Improvements in Transient Stability Using FACTS Devices

    This project focuses on enhancing transient stability in power systems using FACTS (Flexible AC Transmission Systems). It employs MATLAB Simulink to model and analyze the IEEE 9-Bus System, implementing controllers like SVC (Static VAR Compensator) and STATCOM (Static Synchronous Compensator). These devices help regulate voltage stability, optimize power flow, and improve overall system resilience against disturbances.

    All project-related files and resources are available on GitHub.
